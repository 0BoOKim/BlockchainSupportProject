# BlockchainSupportProject

Hyperledger Fabric을 이용한 개인 후원 시스템 by 이민주, 유소영, 하유진

## 1. 환경설정 
### 1-1. 우분투 설치하기

- 필요한 프로그램
```
버추얼 박스 (virtual box)
```
    
우선 버추얼 박스와 우분투의 iso 파일을 다운로드한다. 우분투는 16.04 이상을 사용해야한다. 다운로드를 모두 마쳤다면 버추얼 박스를 실행시키고 새 가상머신을 생성한다. 가상머신의 종류는 리눅스, 버전은 다운로드한 우분투 iso의 비트에 해당하는 Ubuntu로 설정합니다. 메모리의 크기는 2048MB 이상, 하드 디스크의 크기는 35GB 이상으로 설정해주어야 한다.
